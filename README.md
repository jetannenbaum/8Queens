# 8Queens
A recursive solution to the 8 queens problem.
Place 8 queens on an 8 X 8 chessboard so they can't attack each other.
A potential soulution is:
   0 1 2 3 4 5 6 7
0: Q - - - - - - -
1: - - - - Q - - -
2: - - - - - - - Q
3: - - - - - Q - -
4: - - Q - - - - -
5: - - - - - - Q -
6: - Q - - - - - -
7: - - - Q - - - -
